The COVID-19 vaccine benefit-risk assessment tool facilitates the quantification of the benefits and the risks associated with various vaccines.

The tool can be used to model the benefits (prevented cases, hospitalizations, icu admissions and deaths) and risks of various COVID-19 vaccines.
By uploading the required data sets the user can chose to model the benefits for:
  • a set of European countries
  • a set of age categories
  • a range of vaccines
  • a number of variants of concern
Benefits are computed independent of sex and a maximum of 3 vaccine doses are allowed.

By uploading the required data sets the user can chose to model the risks for:
  • a set of European countries
  • males and females separately
  • a range of vaccines
  • a list of conditions which can differ per vaccine
  • a set of age categories which differ per condition
  • the background risk rate to compare the results to


Running the tool requires the following information:
  • For each country and age group included in the benefits analysis:
    • Number of inhabitants
    • Proportion of this age group in the total COVID-19 burden for cases, hospitalizations, icu admission and mortality
  • For each country and date included in the benefits analysis:
    • Number of COVID-19 cases, hospital admissions, icu admissions and deaths
  • For each country, date and variant of concern included in the benefits analysis:
    • Proportion of the COVID-19 infections with this variant
  • For each country, date, age group and vaccine included in the benefit analysis:
    • Number of first, second and third doses administred
  • For each age group, sex, condition and vaccine included in the risk analysis:
    • Number of observed risk cases aggregated over the whole study period
  • For each age group, sex and condition included in the risk analysis:
    • Number of observed risk cases not induced by vaccination
    • Number of observed person years
    Various data sources and observation periods can be included in the data.
  • For each age group, sex and vaccine in the risk analysis:
    • The number of administred vaccines over the whole study period
    When this number is imputed also the uncertainty of this imputation is required.
